## 🚀 Overview

Full end-to-end data analyst capstone: data collection → wrangling → EDA
→ visualization, analysing 65,000 developer survey responses.

## 📂 Project Structure

-   data/ -- datasets\
-   notebooks/ -- analysis notebooks\
-   visuals/ -- charts\
-   README.md -- documentation

## 🛠️ Tools & Technologies

-   Python (Pandas, NumPy)
-   Matplotlib, Seaborn
-   Jupyter Notebook

## 🔍 Workflow

1.  Data Collection\
2.  Data Cleaning\
3.  Exploratory Data Analysis\
4.  Visualization
